# HeartDieseasePred

Heart disease remains one of the leading causes of mortality worldwide, making early detection critical for reducing fatal outcomes. This study presents an end-to-end machine learning framework for predicting the presence of heart disease using clinical and demographic data. A dataset consisting of 180 patient records with 15 attributes was analyzed, preprocessed, and modeled using multiple classification algorithms. Logistic Regression, Random Forest, Decision Tree, K-Nearest Neighbors (KNN), and XGBoost were evaluated and tuned using RandomizedSearchCV. Among all models, Logistic Regression achieved the highest predictive performance with an accuracy of 86.1% and a ROC–AUC score of 0.90. Model interpretability techniques, including SHAP and permutation importance, were applied to identify clinically meaningful predictors. The final trained pipeline was serialized for deployment, demonstrating the feasibility of applying machine learning to real-world heart disease risk prediction.
